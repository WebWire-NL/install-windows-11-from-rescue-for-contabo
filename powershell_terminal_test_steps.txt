# Steps to Verify PowerShell Terminal

1. **Open a New PowerShell Terminal**:
   - Command: `powershell`
   - This opens a new PowerShell terminal.

2. **Test the Terminal**:
   - Command: `echo Hello`
   - This prints `Hello` to the terminal to verify it is working.

3. **Expected Output**:
   - The terminal should display `Hello`.

4. **Troubleshooting**:
   - If the output is not visible, ensure the terminal is active and not hidden.
   - Restart VS Code and repeat the steps if necessary.

5. **Git Branch Correction**:
   - We made a mistake by switching to `master` or `main`.
   - Ensure we are on the original branch where the work was being done.
   - Command: `git checkout <original-branch-name>`

6. **Push Changes to Remote**:
   - After verifying the branch, push the changes to the remote repository.
   - Commands:
     - `git add .`
     - `git commit -m "Your commit message"`
     - `git push origin <original-branch-name>`

7. **Next Steps**:
